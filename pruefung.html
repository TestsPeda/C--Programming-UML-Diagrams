Probe-SA

Zwei Teile, 30 Minuten Zeit: ein Auswahlteil (15 Fragen, 15 P, automatisch ausgewertet) und ein offener Teil (Programmieren & Diagramm, 13 P, von einer KI bewertet) — gesamt 28 Punkte. Auswahlteil am Ende abgeben; für den offenen Teil den KI-Bewertungs-Prompt kopieren. Ergebnisse werden lokal gespeichert.

Verbleibend 30:00
01Grundlagen1 P

Welchen Datentyp liefert Console.ReadLine()?

  • int
  • string
  • double
  • den passenden Typ automatisch
Lösung

string — vor dem Rechnen konvertieren (Convert.ToInt32(...)).

02Grundlagen1 P

Was ergibt 7 / 2 bei zwei int-Operanden?

  • 3.5
  • 3
  • 4
  • Kompilierfehler
Lösung

Ganzzahldivision → 3 (Nachkomma fällt weg). Für 3.5 muss ein Operand double sein.

03Grundlagen1 P

Welcher Typ passt am besten für einen Geldbetrag?

  • int
  • double
  • decimal
  • string
Lösung

decimal — dezimal-genau, keine Rundungsfehler (Suffix m).

04Grundlagen1 P

Welche Schleife passt, wenn die Anzahl der Wiederholungen vorab feststeht?

  • for
  • while
  • do-while
  • switch
Lösung

for — Startwert, Bedingung und Schritt in einer Zeile.

05Grundlagen1 P

Wozu dient break; in einem switch?

  • beendet das Programm
  • verlässt den switch, kein Durchfallen in den nächsten case
  • springt zum Schleifenanfang
  • hat keine Wirkung
Lösung

Es verlässt den switch nach dem passenden case.

06Grundlagen1 P

Wie prüft man, ob z ein Vielfaches von 5 ist?

  • z / 5 == 0
  • z % 5 == 0
  • z % 5 == 1
  • z * 5 == 0
Lösung

z % 5 == 0 — der Rest ist 0.

07Arrays1 P

Welchen Index hat das erste Element eines Arrays?

  • 1
  • 0
  • -1
  • frei wählbar
Lösung

0-basiert: erstes Element [0], letztes [Length-1].

08Arrays1 P

Wie deklariert man ein 2D-Array mit 3 Zeilen und 4 Spalten (string)?

  • string[] m = new string[3][4];
  • string[,] m = new string[3, 4];
  • string[3,4] m;
  • string m = new string[3, 4];
Lösung

Komma-Syntax: string[,] und new string[3, 4].

09Arrays1 P

Womit ermittelt man die Anzahl der Spalten eines 2D-Arrays m?

  • m.Length
  • m.GetLength(0)
  • m.GetLength(1)
  • m.Columns
Lösung

GetLength(1) = Spalten, GetLength(0) = Zeilen.

10Arrays1 P

Wie viele Plätze hat int[,] saal = new int[10, 15]; insgesamt?

  • 25
  • 10
  • 150
  • 15
Lösung

10 × 15 = 150 (Zeilen × Spalten).

11Arrays1 P

Eingabe Zeile 1, Spalte 1 (1-basiert). Welcher Array-Zugriff ist richtig?

  • lager[1, 1]
  • lager[0, 0]
  • lager[1][1]
  • lager(0, 0)
Lösung

1-basiert minus 1 → [0, 0]; 2D mit Komma.

12UML1 P

Womit wird eine Aktion dargestellt?

  • Raute
  • abgerundetes Rechteck
  • ausgefüllter Kreis
  • dicker Balken
Lösung

Abgerundetes Rechteck mit dem Tätigkeitsnamen.

13UML1 P

Wofür steht der ausgefüllte Kreis mit Ring (◉)?

  • Startpunkt
  • Endpunkt der Aktivität
  • Entscheidungsknoten
  • Gabelung
Lösung

Endpunkt. Der Start ist der ausgefüllte Kreis ohne Ring.

14UML1 P

Was bewirkt eine Gabelung (Balken)?

  • teilt den Fluss in parallele Teilflüsse
  • wählt je Bedingung genau einen Pfad
  • beendet die Aktivität
  • kennzeichnet eine Bedingung
Lösung

Gabelung = parallel (sowohl als auch). Die Entscheidung (Raute) = entweder/oder.

15UML1 P

Eine Vereinigung nach einer Verzweigung entspricht im Code …

  • der gemeinsamen Anweisung nach dem if-else
  • einer zusätzlichen Schleife
  • dem Programmende
  • einem zweiten if im ersten
Lösung

Die erste Anweisung hinter dem if-else — läuft unabhängig vom Pfad.

Offener Teil — wird von einer KI bewertet

Diese Aufgaben werden nicht automatisch ausgewertet. Löse sie (Code eintippen bzw. Diagramm auf Papier zeichnen), kopiere dann den KI-Bewertungs-Prompt — einzeln oder gesammelt — und lass eine KI die Punkte vergeben. Max. 13 Punkte.

P1ProgrammierenVerzweigung4 P · KI

Schreiben Sie ein Programm: Lesen Sie den Bestellwert ein. Liegt er unter 100,00 €, addieren Sie eine Versandpauschale von 6,50 €; ab 100,00 € keine Versandkosten. Geben Sie den Gesamtbetrag aus.

Musterlösung 
using System;

Console.Write("Bestellwert in EUR: ");
double wert = Convert.ToDouble(Console.ReadLine());

double gesamt = wert;
if (wert < 100.0)
    gesamt = wert + 6.5;

Console.WriteLine($"Gesamtbetrag: {gesamt} EUR");
P2ProgrammierenArrays 2D4 P · KI

Legen Sie ein zweidimensionales Array int[2,3] mit beliebigen Werten an. Geben Sie das Element in Zeile 2, Spalte 1 (so wie der Mensch zählt, 1-basiert) auf der Konsole aus.

Musterlösung 
using System;

int[,] werte = new int[2, 3]
{
    { 5, 8, 3 },
    { 7, 1, 9 }
};

// Zeile 2, Spalte 1 (1-basiert) -> Index [1, 0]
Console.WriteLine(werte[1, 0]);   // 7
P3Diagramm zeichnenUML5 P · KI

Auf Papier zeichnen. Erstellen Sie ein UML-Aktivitätsdiagramm für den Login: Der Benutzer gibt ein Passwort ein. Ist es falsch, wird erneut gefragt — höchstens 3 Versuche. Nach dem 3. Fehlversuch wird das Konto gesperrt (Ende). Bei richtigem Passwort wird die Startseite angezeigt.

Beschreiben Sie Ihr gezeichnetes Diagramm unten (Elemente + Reihenfolge) — oder hängen Sie der KI ein Foto des Blattes an.

Musterlösung (Elemente)
  1. Startpunkt ● → Aktion „Passwort eingeben"
  2. Entscheidung „Passwort korrekt?" mit [ja] / [nein]
  3. [ja] → Aktion „Startseite anzeigen" → Endpunkt ◉
  4. [nein] → Entscheidung „3. Versuch?"
  5. [nein] (noch Versuche) → Rückführung (Schleife) zu „Passwort eingeben"
  6. [ja] (3. Fehlversuch) → Aktion „Konto sperren" → Endpunkt ◉ (bzw. Flussende ⊗)

Entspricht dem Muster von Aufgabe 12 (Geldautomat) auf der UML-Seite: Schleife + verschachtelte Entscheidung + Abschluss.

So funktioniert die Bewertung

Der Auswahlteil (15 P) wird beim Abgeben automatisch ausgewertet. Der offene Teil (13 P) wird nicht vom Programm bewertet: Kopiere den KI-Bewertungs-Prompt (je Aufgabe oder gesammelt) und lasse eine KI (z. B. ChatGPT/Claude) die Punkte nach den angegebenen Kriterien vergeben. Gesamtpunktzahl = Auswahlteil + offener Teil (max. 28 P).
