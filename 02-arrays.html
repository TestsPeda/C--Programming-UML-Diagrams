Eindimensionale Arrays

Ein Array ist eine nummerierte Reihe von Speicherplätzen gleichen Typs — die Zählung beginnt bei 0.

01Multiple ChoiceAuto

Welchen Index hat das erste Element eines C#-Arrays?

  • 1
  • 0
  • -1
  • frei wählbar
Lösung

Arrays sind 0-basiert: Das erste Element steht bei Index 0, das letzte bei Length - 1.

02Multiple ChoiceAuto

Wie deklariert man ein Array für 6 Lottozahlen (int) korrekt?

  • int lotto = new int[6];
  • int[] lotto = new int[6];
  • int[6] lotto = new int;
  • array<int> lotto = 6;
Lösung

Schema: Typ[] name = new Typ[Anzahl];. Die eckigen Klammern hinter dem Typ kennzeichnen ein Array; die Größe steht hinter new. Alternativ mit Initialisierung: int[] lotto = {7, 12, 23, 31, 42, 49};.

03Multiple ChoiceAuto

Das Array int[] lotto hat 6 Elemente. Welcher Zugriff führt zu einer IndexOutOfRangeException?

  • lotto[0]
  • lotto[5]
  • lotto[6]
  • lotto[lotto.Length - 1]
Lösung

Gültige Indizes sind 0 bis 5 (= Length - 1). lotto[6] liegt außerhalb → Laufzeitfehler. Klassiker Verwechslung von Anzahl (6) und letztem Index (5).

04Multiple ChoiceAuto

Mit welcher Schleife durchläuft man ein eindimensionales Array über seine Indizes von Anfang bis Ende?

  • for (int i = 0; i < lotto.Length; i++)
  • for (int i = 1; i <= lotto.Length; i++)
  • for (int i = 0; i <= lotto.Length; i++)
  • for (int i = lotto.Length; i > 0; i--)
Lösung

Start bei 0, Bedingung i < Length (nicht <=!), Schritt i++. Mit <= würde man auf Index Length zugreifen → IndexOutOfRangeException. Alternativ ohne Index: foreach (int zahl in lotto).

05Multi-SelectAuto

Welche Aussagen über eindimensionale Arrays sind richtig? (mehrere möglich)

  • Alle Elemente haben denselben Datentyp.
  • Length liefert die Anzahl der Elemente.
  • Die Größe eines Arrays lässt sich nach new beliebig vergrößern.
  • Ein int-Array wird standardmäßig mit 0 initialisiert.
  • Der erste Index ist 1.
Lösung

Richtig: 1, 2, 4. Die Größe ist nach new fest (für variable Größe gibt es List<T>), und der erste Index ist 0.

B

Mehrdimensionale Arrays

Eine Tabelle mit Zeilen und Spalten — z. B. ein Kinosaal oder ein Lagerregal.

06Multiple ChoiceAuto

Wie deklariert man ein zweidimensionales string-Array mit 3 Zeilen und 4 Spalten?

  • string[] lager = new string[3][4];
  • string[,] lager = new string[3, 4];
  • string[3,4] lager;
  • string lager = new string[3, 4];
Lösung

Bei einem rechteckigen 2D-Array steht ein Komma in den Klammern: string[,] und new string[3, 4]. Die Schreibweise [3][4] wäre ein „jagged array" (Array aus Arrays) — anderes Konzept.

07Multiple ChoiceAuto

Ein Kinosaal ist als int[,] saal = new int[10, 15]; angelegt. Wie viele Plätze hat der Saal insgesamt?

  • 25
  • 10
  • 150
  • 15
Lösung

Plätze = Zeilen × Spalten = 10 × 15 = 150. saal.Length liefert hier die Gesamtzahl 150, saal.GetLength(0) = 10 (Zeilen), saal.GetLength(1) = 15 (Spalten).

08Multiple ChoiceAuto

Mit welchem Befehl ermittelt man die Anzahl der Spalten eines 2D-Arrays lager?

  • lager.Length
  • lager.GetLength(0)
  • lager.GetLength(1)
  • lager.Columns
Lösung

GetLength(0) = 1. Dimension (Zeilen), GetLength(1) = 2. Dimension (Spalten). Length liefert dagegen die Gesamtzahl aller Elemente.

09Zuordnungmanuell

Der Benutzer gibt im Chaos-Lager Zeile 2, Spalte 3 (1-basiert, wie der Mensch zählt) ein. Welcher Array-Index gehört dazu, und warum?

Musterlösung
Eingabe (Mensch): Zeile = 2, Spalte = 3 // 1-basiert Array-Index: lager[2 - 1, 3 - 1] = lager[1, 2] // 0-basiert

Da Arrays bei 0 beginnen, der Mensch aber bei 1 zählt, zieht man von jeder Eingabe 1 ab. Das ist die zentrale Indexumrechnung der Sheldon-Aufgabe.

Die mehrdimensionale Array-Aufgabe „Mit Amazon durchs Chaos-Lager" als vollständige Programmieraufgabe.

10ProgrammierenSelbsttest

Chaos-Lager. Erfassen Sie die Produkte eines Lagers (3 Zeilen × 4 Spalten) in einem mehrdimensionalen Array. Der Benutzer gibt eine Zeile (1–3) und Spalte (1–4) ein; das Programm gibt das zugehörige Produkt aus. Für Schnelle: Wird ein Produkt aus der ersten Zeile gewählt, soll zusätzlich „Produkte im Angebot" erscheinen.

Tipp: Code oben eintippen, dann „KI-Bewertungs-Prompt kopieren" → in ChatGPT/Claude einfügen für Feedback. Die Seite selbst führt keinen Code aus.

Musterlösung 
using System;

// Chaos-Lager: 3 Zeilen x 4 Spalten
string[,] lager = new string[3, 4]
{
    { "Laptop",   "Drucker", "PC",     "Tablet" },
    { "MP3-Player","iPhone", "Galaxy", "-"      },
    { "Smart-TV", "Router",  "-",      "-"      }
};

// Eingabe (1-basiert, wie der Mensch zählt)
Console.Write("Zeile (1-3): ");
int zeile = Convert.ToInt32(Console.ReadLine());
Console.Write("Spalte (1-4): ");
int spalte = Convert.ToInt32(Console.ReadLine());

// Indexumrechnung: Array ist 0-basiert -> minus 1
string produkt = lager[zeile - 1, spalte - 1];
Console.WriteLine($"Produkt: {produkt}");

// Für Schnelle: erste Zeile = Angebot
if (zeile == 1)
    Console.WriteLine("Produkte im Angebot");
Bewertungs-Checkliste
  • 2D-Array string[,] korrekt deklariert & mit 3×4 Produkten befüllt
  • Zeile (1–3) und Spalte (1–4) eingelesen und nach int konvertiert
  • Indexumrechnung lager[zeile-1, spalte-1] korrekt
  • zugehöriges Produkt korrekt ausgegeben
  • Für-Schnelle: erste Zeile → „Produkte im Angebot"
  • strukturierte Darstellung & sinnvolle Kommentare

Hinweis Robuster wäre zusätzlich eine Prüfung, ob zeile in 1–3 und spalte in 1–4 liegen, sonst droht eine IndexOutOfRangeException.

11Multiple ChoiceAuto

Im Chaos-Lager gibt der Benutzer Zeile 1 und Spalte 1 ein. Welcher Array-Zugriff liefert das richtige Produkt?

  • lager[1, 1]
  • lager[0, 0]
  • lager[1][1]
  • lager(0, 0)
Lösung

Eingabe 1/1 (1-basiert) → Index [0, 0] (0-basiert). Bei 2D-Arrays mit Komma-Syntax: lager[zeile, spalte], nicht lager[..][..].

12Multiple ChoiceAuto

Was ist ein jagged array (int[][]) im Unterschied zum rechteckigen int[,]?

  • Beides ist identisch, nur andere Schreibweise.
  • Ein „Array aus Arrays", dessen Zeilen unterschiedlich lang sein dürfen.
  • Ein Array, das nur Zahlen speichern kann.
  • Ein Array ohne Index.
Lösung

Das jagged array int[][] ist ein Array, dessen Elemente selbst Arrays sind — jede „Zeile" kann eine andere Länge haben. Das rechteckige int[,] ist dagegen immer ein festes Gitter Zeilen × Spalten.

13Szenariomanuell

Wie durchläuft man alle Plätze eines 2D-Saals int[,] saal und zählt die belegten (Wert 1)? Skizzieren Sie die verschachtelte Schleife.

Musterlösung
int belegt = 0; for (int z = 0; z < saal.GetLength(0); z++) // Zeilen for (int s = 0; s < saal.GetLength(1); s++) // Spalten if (saal[z, s] == 1) belegt++;

Die äußere Schleife läuft über die Zeilen (GetLength(0)), die innere über die Spalten (GetLength(1)). So besucht man jeden der Zeilen×Spalten-Plätze genau einmal.
